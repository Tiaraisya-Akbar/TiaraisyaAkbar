📚 Isi Repositori
Repositori ini berisi kumpulan tugas berbasis website:
📌 Tugas 1 – Website Dasar
✨ Menampilkan informasi sederhana dalam bentuk halaman web:
- Biodata pribadi
- Galeri
- Video
📌 Tugas 2 – Formulir Data
📊 Merancang sistem input data berbasis web:
Data Demografi (nama, tanggal lahir, alamat, dll)
Data Bank (nama bank & nomor rekening)
Data Transaksi Keuangan (tanggal, jenis transaksi, debit & kredit)

🎯 Tujuan Pembuatan
Repositori ini dibuat untuk memenuhi tugas mata kuliah Sistem Informasi Manajemen Perikanan, dengan tujuan:
- Mengembangkan kemampuan dasar dalam pembuatan website
- Memahami pengelolaan dan penyajian data berbasis sistem informasi
- Menerapkan formulir sebagai alat input data digital
- Melatih keterampilan dalam merancang tampilan informasi yang rapi dan terstruktur
- Mengintegrasikan teknologi informasi dalam bidang Sosial Ekonomi Perikanan
